BasicBarChart Class

Defined in: dynamic_chart.js:493
Module: DynamicChart

Base Chart Class: BasicBarChart
Basic bar histogram chart.
Values are represented using vertical bars;
Each point or value can have up to 10 subcomponents, where each component can be any non-nregative real number (i.e., each point can be in R_+^i, for 1 <= i <= 10).
